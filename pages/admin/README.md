# 初始化题目数据

## 使用方法

1. **访问初始化页面**
   - 在小程序中导航到：`pages/admin/init-questions`
   - 或者直接访问路径：`/pages/admin/init-questions`

2. **点击初始化按钮**
   - 页面会显示题目总数（约995题）
   - 点击"开始初始化"按钮
   - 确认操作后开始初始化

3. **等待完成**
   - 初始化过程可能需要几分钟
   - 页面会显示实时状态和结果
   - 完成后会显示统计信息

## 注意事项

1. **数据库权限**
   - 初始化前需要确保数据库 schema 允许创建操作
   - 文件：`uniCloud-aliyun/database/questions_master.schema.json`
   - 需要设置 `"create": true`

2. **登录要求**
   - 初始化需要登录状态
   - 确保已登录后再访问页面

3. **去重处理**
   - 已存在的题目不会重复插入
   - 可以安全地多次运行初始化

4. **初始化完成后**
   - 可以将 schema 的 `create` 权限改回 `false`（可选，提高安全性）
   - 现在可以正常使用评估功能了

## 故障排除

如果初始化失败：
1. 检查是否已登录
2. 检查数据库权限设置
3. 查看控制台日志获取详细错误信息
4. 确认云函数 `initQuestions` 已正确部署
