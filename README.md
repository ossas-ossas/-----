# 知动儿童综合发育测评 - 微信小程序

## 项目简介

这是一个基于 uni-app (Vue2) 框架开发的儿童发育评估微信小程序，采用 pastel 柔和配色方案，界面简洁易读，适合家长用户使用。

## 功能特点

- 🎯 **科学评估**：基于专业发育量表进行综合评估
- 📊 **详细报告**：生成个性化的发育分析报告
- 📈 **趋势追踪**：记录历史评估，追踪发育趋势
- 🎨 **友好界面**：使用 pastel 柔和配色，界面简洁美观
- 📱 **移动优先**：专为微信小程序优化，操作便捷

## 页面结构

### 1. 首页 (`pages/index/index.vue`)
- **功能**：应用介绍和入口页面
- **内容**：
  - 应用标题："知动儿童综合发育测评"
  - 功能介绍：填写信息、完成评估、获得报告
  - 主按钮：【开始评估】→ 跳转到儿童信息页
- **设计**：渐变背景、装饰圆圈、功能介绍卡片

### 2. 儿童信息填写页 (`pages/child-info/child-info.vue`)
- **功能**：收集儿童基本信息和家庭信息
- **表单字段**：
  - 基本信息：姓名、性别、出生日期
  - 家庭信息：主要照顾者、联系电话
  - 发育相关：发育迟缓史、备注信息
- **验证**：实时表单验证，确保必填信息完整
- **年龄计算**：自动计算并显示儿童年龄

### 3. 评估量表页 (`pages/assessment/assessment.vue`)
- **功能**：进行发育评估测试
- **评估维度**：
  - 大运动：独立坐稳能力
  - 精细动作：拇指食指捏取能力
  - 语言理解：简单指令理解
  - 语言表达：词语表达能力
  - 社交行为：主动互动能力
  - 认知能力：动作模仿能力
  - 自理能力：独立进食能力
  - 情绪调节：情绪稳定性
- **交互**：进度条显示、选项选择、前后翻页
- **评分**：每题1-3分，自动计算总分

### 4. 结果报告页 (`pages/result/result.vue`)
- **功能**：展示详细的评估结果和建议
- **报告内容**：
  - 总体评分：百分制评分和等级
  - 详细分析：各维度得分和描述
  - 建议指导：个性化建议和指导
  - 评估详情：评估时间、年龄、得分等
- **操作**：查看历史记录、重新评估

### 5. 历史记录页 (`pages/history/history.vue`)
- **功能**：管理历史评估记录
- **功能特点**：
  - 统计概览：总评估次数、最新得分、平均得分
  - 记录列表：按时间倒序显示历史记录
  - 趋势分析：可视化发育趋势图表
  - 记录管理：查看详情、清空记录
- **空状态**：无记录时的引导界面

## 技术架构

### 框架和工具
- **框架**：uni-app (Vue2)
- **平台**：微信小程序
- **样式**：SCSS + 响应式设计
- **存储**：uni.storage 本地存储

### 配色方案
采用 pastel 柔和配色，适合儿童相关应用：

- **主色调**：天空蓝 (#87CEEB)
- **辅助色**：薄荷绿 (#98FB98)
- **警告色**：珊瑚粉 (#FFB6C1)
- **错误色**：柔和红 (#FFA0A0)
- **背景色**：浅蓝渐变 (#E8F4FD → #F0F8FF)

### 数据结构

#### 儿童信息
```javascript
{
  name: 'string',           // 姓名
  gender: 'male|female',    // 性别
  birthDate: 'YYYY-MM-DD',  // 出生日期
  caregiver: 'string',       // 主要照顾者
  phone: 'string',          // 联系电话
  hasDelayHistory: 'yes|no', // 发育迟缓史

```

#### 评估结果
```javascript
{

  answers: [],              // 答案数组

  maxScore: number,         // 满分
  scorePercent: number,     // 得分百分比
```

## 使用说明

### 开发环境
1. 安装 HBuilderX
2. 导入项目
3. 配置微信小程序 AppID
4. 运行到微信开发者工具

### 功能流程
1. **开始评估**：首页点击"开始评估"按钮
2. **填写信息**：在儿童信息页填写基本信息
3. **进行评估**：完成8个维度的评估测试
4. **查看报告**：获得详细的评估结果和建议
5. **管理记录**：在历史记录页查看和管理过往评估

### 注意事项
- 本评估结果仅供参考，不能替代专业医疗诊断
- 建议定期进行评估，追踪发育趋势
- 如有疑问，请咨询专业医生
