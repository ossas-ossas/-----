# 知动儿童综合发育测评 - 微信小程序

## 项目简介

这是一个基于 uni-app (Vue2) 框架开发的儿童发育评估微信小程序，采用 pastel 柔和配色方案，界面简洁易读，适合家长用户使用。

## 功能特点

- 🎯 **专业量表**：基于"低张儿综合功能发育量表"的专业评估
- 📅 **分龄评估**：按年龄段(1个月-6岁)提供针对性评估项目
- 📊 **详细报告**：生成个性化的发育分析报告和专业建议
- 📈 **趋势追踪**：记录历史评估，追踪发育趋势
- 🎨 **友好界面**：使用 pastel 柔和配色，界面简洁美观
- 📱 **移动优先**：专为微信小程序优化，操作便捷
- ✅ **极简评估**：1-3个月年龄段采用单选框点击方式，选中变色表示完成

## 页面结构

### 1. 首页 (`pages/index/index.vue`)
- **功能**：应用介绍和入口页面
- **内容**：
  - 应用标题："知动儿童综合发育测评"
  - 功能介绍：填写信息、完成评估、获得报告
  - 主按钮：【开始评估】→ 跳转到儿童信息页
- **设计**：渐变背景、装饰圆圈、功能介绍卡片

### 2. 儿童信息填写页 (`pages/child-info/child-info.vue`)
- **功能**：收集儿童基本信息和家庭信息
- **表单字段**：
  - 基本信息：姓名、性别、出生日期
  - 家庭信息：主要照顾者、联系电话
  - 发育相关：发育迟缓史、备注信息
- **验证**：实时表单验证，确保必填信息完整
- **年龄计算**：自动计算并显示儿童年龄

### 3. 评估量表页 (`pages/assessment/assessment.vue`)
- **功能**：进行专业发育评估测试
- **分龄评估**：根据儿童年龄自动匹配对应年龄段的评估项目
- **评估维度**：
  - **外在感知觉(OS)功能发育**：视知觉、听知觉、触知觉
  - **内在感知觉(IS)功能发育**：本体运动知觉、前庭平衡知觉
  - **躯肢体粗大动作(GM)功能发育**：基本动作、高级动作
  - **双上肢粗大动作(UE)功能发育**：视动协调、操作技能
  - **精细动作(FM)功能发育**：手部精细操作
  - **社会性(SI)互动功能发育**：与人互动、情绪与自我
  - **认知(M)功能发育**：注意、记忆、概念、推理
  - **口腔动作(OM)功能发育**：口腔协调能力
  - **语言(ML)功能发育**：口语理解、口语表达
- **年龄段**：1-3个月、3-6个月、6-9个月、9-12个月、1-1.5岁、1.5-2岁、2-2.5岁、2.5-3岁、3-4岁、4-5岁、5-6岁
- **交互**：进度条显示、单选框点击、前后翻页
- **评估方式**：1-3个月年龄段采用极简单选框，点击变色表示完成，未选择默认为未完成

### 4. 结果报告页 (`pages/result/result.vue`)
- **功能**：展示详细的评估结果和专业建议
- **报告内容**：
  - 总体评分：百分制评分和等级
  - 年龄段信息：显示评估的年龄段
  - 详细分析：各维度得分和描述
  - 专业建议：基于年龄段的个性化建议
  - 评估详情：评估时间、年龄、得分等
- **专业建议**：根据年龄段和得分提供针对性的康复建议
- **操作**：查看历史记录、重新评估

### 5. 历史记录页 (`pages/history/history.vue`)
- **功能**：管理历史评估记录
- **功能特点**：
  - 统计概览：总评估次数、最新得分、平均得分
  - 记录列表：按时间倒序显示历史记录
  - 趋势分析：可视化发育趋势图表
  - 记录管理：查看详情、清空记录
- **空状态**：无记录时的引导界面

## 技术架构

### 框架和工具
- **框架**：uni-app (Vue2)
- **平台**：微信小程序
- **样式**：SCSS + 响应式设计
- **存储**：uni.storage 本地存储
- **数据源**：基于"低张儿综合功能发育量表"的专业题库

### 配色方案
采用 pastel 柔和配色，适合儿童相关应用：

- **主色调**：天空蓝 (#87CEEB)
- **辅助色**：薄荷绿 (#98FB98)
- **警告色**：珊瑚粉 (#FFB6C1)
- **错误色**：柔和红 (#FFA0A0)
- **背景色**：浅蓝渐变 (#E8F4FD → #F0F8FF)

## 使用说明

### 开发环境
1. 安装 HBuilderX
2. 导入项目
3. 配置微信小程序 AppID
4. 运行到微信开发者工具

### 功能流程
1. **开始评估**：首页点击"开始评估"按钮
2. **填写信息**：在儿童信息页填写基本信息，系统自动计算年龄段
3. **进行评估**：根据年龄段完成对应的专业评估测试
4. **查看报告**：获得详细的评估结果和专业建议
5. **管理记录**：在历史记录页查看和管理过往评估

### 注意事项
- 本评估基于"低张儿综合功能发育量表"，结果仅供参考，不能替代专业医疗诊断
- 建议定期进行评估，追踪发育趋势
- 如有疑问，请咨询专业医生或康复师
- 早期干预效果最佳，发现问题请及时寻求专业帮助

### 调试步骤
如果遇到题库加载问题，请检查：
1. 打开微信开发者工具的控制台（Console）
2. 在儿童信息页面填写信息后，点击"开始评估"
3. 查看控制台输出的调试信息：
   - 儿童信息
   - 计算出的年龄（月）
   - 获取到的题库数据
   - 最终问题数量
4. 确认年龄计算和题库匹配是否正确
